📹 カメラアクセス設定ガイド

AI動作分析でカメラを使用するための設定方法

🌐 方法1: HTTPS接続を使用（推奨）

最も確実で安全な方法です。

1 ngrok を使用してHTTPS接続を作成:
npm install -g ngrok
ngrok http 3001

表示されたHTTPS URL（例: https://abc123.ngrok.io）を使用してアクセス

2 または、ローカルホスト接続:
localhost で開く
🔧 方法2: Chromeの設定を変更

開発用途でHTTP接続でもカメラアクセスを許可します。

1 Chrome フラグを設定:
chrome://flags/#unsafely-treat-insecure-origin-as-secure
2 以下を入力:
http://192.168.3.186:3001
3 「Enabled」に設定して再起動
⚠️ 注意: この設定は開発用途のみで使用し、一般的なブラウジングでは無効にしてください。
📱 方法3: 動画ファイルを使用

カメラが使用できない場合の代替案

1 スマートフォンで運動動画を撮影
2 動画ファイルをアップロード
動画アップロードページへ
3 サーバーAI分析で詳細評価

✅ 推奨環境

  • ブラウザ: Chrome 90+, Firefox 88+, Safari 14+
  • 接続: HTTPS または localhost
  • カメラ: 内蔵カメラまたは外付けWebカメラ
  • 解像度: 640x480以上
🎥 動画AI分析に戻る 📋 デモメニューに戻る

🛠 トラブルシューティング

  • カメラアクセス拒否: ブラウザのアドレスバー左端のカメラアイコンをクリックして許可
  • カメラが見つからない: 他のアプリケーションがカメラを使用していないか確認
  • 画質が悪い: 明るい場所で撮影し、カメラレンズを清掃
  • 処理が重い: ブラウザを再起動するか、他のタブを閉じる
